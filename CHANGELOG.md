# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-21
### Added
- Initial public release of `LianLi_HydroShift_LCD_RGB` plugin.
- Stable LED control paths for HydroShift (`VID 0x0416`, `PID 0x7399`).
- Support for SignalRGB modes: `Forced`, `Canvas`, `Paint`, `Depaint`, `Shutdown`.
- Public documentation, license, and installation paths (Add-on URL + manual fallback).

### Notes
- Scope is LED-only.
- LCD, pump, and fan speed/RPM controls are intentionally out of scope.
- Screenshot assets currently include placeholders and should be replaced by real captures.
